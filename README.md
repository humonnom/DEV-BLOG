## 블로그

   - 미니프로젝트 배포
   - 최적화, UX/UI 공부 및 연구

### 사용한 기술
   - Next.js
   - Firebase

### 카테고리 설명
   - HOME: 메인 페이지
   - BLOG: 카테고리 별로 분류된 게시글 목록

   - CONTACT: 연락처, 메일 등의 정보와 메세지를 남길 수 있는 폼으로 구성
